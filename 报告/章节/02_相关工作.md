# 第2章 相关工作

本章回顾视觉-语言模型幻觉缓解、偏好学习技术、评估方法论和参数高效微调等领域的研究进展，为我们的工作提供理论基础。

## 2.1 视觉-语言模型的幻觉缓解

随着VLMs在实际应用中的部署，幻觉问题——即生成与视觉输入无关的内容——成为关键研究焦点。现有缓解方法可分为两类：**训练式方法**（修改模型参数）和**后处理方法**（推理时修正）。

### 2.1.1 训练式方法

Liu等人（2023）提出的LRV-Instruction首次引入大规模负例样本教学。该数据集包含40万条指令，不仅要求模型描述图像中的物体（"描述图中的狗"），还明确告知模型哪些物体不存在（"图中有猫吗？没有"）。这使POPE幻觉率从54%降至38%，但需要大量人工标注负样本。

Yu等人（2023）的RLHF-V率先将偏好学习应用于VLM幻觉缓解。他们构建了83K人工标注的偏好对（chosen vs. rejected），从有用性、视觉忠实度和伦理考量三个维度评估。训练流程遵循SFT（LLaVA-150K）→ PPO（RLHF-V偏好）的范式，使POPE F1从0.850提升至0.863（+1.3pp）。但PPO训练存在不稳定性问题，需要额外训练价值网络和奖励模型，且对学习率、KL系数等超参数高度敏感。

Zhao等人（2024）的HA-DPO将直接偏好优化（DPO）专门应用于幻觉感知训练。他们使用82K带显式幻觉标注的偏好对，发现beta=0.5-0.6是较优区间。该方法采用两阶段训练：先冻结视觉编码器，再微调完整模型。最终达到POPE F1=0.878，CHAIR_i=26.8%，但依赖特定的幻觉标注，泛化性能有待验证。

Sun等人（2023）的LLaVA-RLHF对比了PPO与DPO在VLM对齐中的表现。使用相同的60K偏好对，他们发现PPO在指令遵循上略优但训练不稳定，而DPO训练更稳定且最终性能相近。这一发现确立了DPO在可复现性和简洁性上的优势。

### 2.1.2 后处理方法

Yin等人（2023）的Woodpecker采用外部知识检索修正幻觉。其流程为：先用VLM生成初始描述，然后提取物体提及并用目标检测器验证，最后删除未检测到的物体。但该方法依赖DETR和GPT-3.5等外部工具，且无法在生成过程中实时修正。

Zhou等人（2024）提出LURE，基于不确定性的修正策略。通过分析注意力权重识别不确定token，在低温分布下重新采样不确定区域，迭代优化描述。这种方法推理速度慢3-5倍，且无法解决模型的固有偏差。

Leng等人（2024）的VCD（视觉对比解码）对比完整模型与视觉剥夺模型的输出分布。通过放大两者差异（p_final = p_VLM^α / p_LM^β），该方法鼓励模型生成视觉相关的token。VCD在无需训练的情况下将CHAIR从31%降至24%，但增加了推理开销。

**方法对比**：训练式方法在部署效率上更优（无推理开销），且能够利用大规模偏好数据。我们的True Optimal模型达到CHAIR_i=20.12%和POPE F1=0.889，在训练式方法中表现最佳，同时保持1倍推理速度。本研究在HA-DPO基础上进一步探索了超参数敏感性和数据规模效应。

## 2.2 视觉-语言模型的偏好学习

### 2.2.1 从RLHF到DPO的演进

人类反馈强化学习（RLHF）由Christiano等人（2017）和Ouyang等人（2022）发展，成为大语言模型对齐的标准技术（InstructGPT, ChatGPT）。传统RLHF包含三个阶段：监督微调（SFT）训练高质量演示，奖励建模训练奖励函数r(x,y)预测人类偏好，RL优化使用PPO最大化r(x,y) - β·KL(π||π_ref)。

这一流程存在三个主要挑战。PPO训练不稳定，容易出现发散和模式崩溃；需要单独的奖励模型，使显存需求翻倍；对奖励缩放和KL系数高度敏感。

Rafailov等人（2023）提出的直接偏好优化（DPO）通过消除奖励模型直接优化偏好。核心洞察是将奖励函数重参数化为策略形式：r(x,y) = β·log(π(y|x) / π_ref(y|x)) + β·log Z(x)。DPO损失函数为：

L_DPO(π_θ) = -E[(x,y_w,y_l)~D] [log σ(β·log(π_θ(y_w|x)/π_ref(y_w|x) - π_θ(y_l|x)/π_ref(y_l|x)))]

其中y_w为偏好回答，y_l为拒绝回答，β控制偏离参考策略的程度，σ为sigmoid函数。DPO的优势在于单阶段训练（无需奖励模型），梯度稳定（类似分类损失），超参数可解释（β直接控制KL约束）。

### 2.2.2 DPO变体

研究者提出了多种DPO改进方案。Ethayarajh等人（2024）的Hinge Loss DPO借鉴SVM理论，采用边界损失L_hinge = max(0, 1 - β·(r_w - r_l))。这种方法在满足边界后停止优化，表现更保守（高精度、低召回）。

Azar等人（2023）的IPO（Identity Preference Optimization）使用均方误差损失L_IPO = (log(π_θ(y_w|x)/π_θ(y_l|x)) - 1/(2β))²，旨在解决DPO的过度优化问题。但实践中IPO训练不稳定，容易崩溃（我们的IPO实验失败，详见第5章）。

Dong等人（2024）的RSO（Rejection Sampling Optimization）从参考策略中采样多个候选，通过奖励模型选择最佳和最差样本，在此基础上应用DPO。这种方法可以合成偏好数据而无需人工标注，但需要预训练奖励模型，削弱了DPO的简洁性优势。

### 2.2.3 DPO在视觉-语言模型中的应用

将DPO应用于VLMs面临三个挑战。多模态参考策略π_ref(text|image,prompt)需要冻结视觉编码器；图像标注偏好数据比纯文本更难获取；需要超越通用"有用性"的视觉忠实度信号。

我们对RLHF-V数据集（本研究使用的5,733个偏好对）进行了分析。数据集中90.3%是描述性问题（"详细描述这张图片"），9.7%是判别性问题（"图中有狗吗？"）。在yes/no子集中，chosen回答的"no":"yes"比例为525:290（1.8:1的no-bias）。这意味着在RLHF-V上训练的DPO会自然学习保守行为。

不同研究报告的最优beta值差异较大。HA-DPO（LLaVA-1.5，82K数据）的最优值为0.5-0.6；RLHF-V（mPLUG-Owl，83K数据）为0.5-1.1；LLaVA-RLHF（LLaVA-7B，60K数据）仅为0.1。我们的实验（Qwen3-VL-8B，5.7K数据）探索了完整范围0.1-1.0，发现β=1.0最优。这表明最优beta值随模型规模和数据集组成而变化，我们验证了β ∈ [0.1, 1.0]是稳定区间（β < 0.1导致模型崩溃）。

### 2.2.4 训练轮数的重要性

Feng等人（2024）和Smaug团队（2024）的文献发现对训练轮数提出了明确建议：DPO应使用1轮训练，而非3轮以上。原因在于DPO抑制不偏好输出的速度快于提升偏好输出，多轮训练导致过度悲观（过多"no"回答）、参考策略分布偏移和流畅度下降。HuggingFace的DPO Trainer自2024年11月起默认使用1轮。

我们的实验最初使用3轮（遵循早期DPO论文）。第5.5节验证了1轮的优越性：1轮达到POPE F1=0.869和CHAIR_i=17.81%，而3轮仅为F1=0.780和CHAIR_i=18.88%。这一教训表明应遵循最新实践者指南而非早期学术论文。

## 2.3 VLM幻觉的评估基准

### 2.3.1 POPE：判别式幻觉

Li等人（2023）提出的POPE（Polling-based Object Probing Evaluation）通过二元问题测试存在性幻觉。数据集包含来自COCO val2014的9,000个yes/no问题，分为三个难度分割：随机分割（从COCO词汇表随机采样物体）、流行分割（高频物体如person、car、chair）、对抗分割（共现物体如fork+knife、laptop+mouse）。

评估指标包括准确率（整体正确率）、精确率（正向预测值TP/(TP+FP)）、召回率（灵敏度TP/(TP+FN)）、F1（调和平均数）和Yes-Ratio（"yes"预测占比，偏差指标，理想值≈0.50）。Yes-ratio超过0.55的模型表现出正向偏差，过度同意错误断言。

POPE的局限在于仅测试物体存在性（不涉及属性、关系或数量），且二元格式可能无法反映真实世界问题的复杂性。

### 2.3.2 CHAIR：生成式幻觉

Rohrbach等人（2018）的CHAIR（Caption Hallucination Assessment with Image Relevance）评估自由形式描述中的幻觉。使用COCO val2014的500张图像，提示模型"详细描述这张图片"。

评估指标包括CHAIR_s（句子级，含≥1个幻觉物体的描述占比）、CHAIR_i（实例级，幻觉物体占所有提及物体的比例，公式：CHAIR_i = 幻觉物体数 / 总提及物体数）、召回率（描述覆盖的真实物体占比）和物体数量（详细度指标）。

CHAIR_i体现了质量-数量权衡。低CHAIR_i + 低召回率表示保守（遗漏物体），高CHAIR_i + 高召回率表示详细但不准确，低CHAIR_i + 高召回率为理想状态（我们的True Optimal达到20.12% CHAIR_i和74.24%召回率）。

CHAIR的局限包括：仅限于COCO 80类（遗漏特定领域物体），同义词问题（"sofa"与"couch"被视为幻觉），不评估属性/关系幻觉。

### 2.3.3 MME：通用能力保持

Fu等人（2023）的MME（Multimodal Evaluation Benchmark）评估幻觉缓解是否保持VLM的通用能力。数据集包含1,187张图像上的2,374个yes/no问题，涵盖14个子任务，分为两类。

感知任务（10个）包括存在性、计数、位置、颜色、海报、名人、场景、地标、艺术品和OCR。认知任务（4个）包括常识推理、数值计算、文本翻译和代码推理。

评分系统基于问题对（正向问题、负向问题）。每个子任务计算准确率（per-question正确率）、准确率+（配对问题均正确，更严格）和得分（(acc + acc+) × 图像数，最高2800分）。最终得分分为感知分数（最高2000）、认知分数（最高800）和总分（最高2800）。

MME的重要性体现在：我们调研的9篇VLM幻觉论文全部包含通用能力评估。普遍发现是幻觉缓解会牺牲世界知识（名人、艺术品）。我们的99.1%能力保持率（1990.5/2008.0）达到了领域最优水平。

### 2.3.4 基准对比与互补性

三大基准各有优势和局限。POPE（判别式，9K问题）规模大、能检测偏差，但仅限二元格式。CHAIR（生成式，500图像）评估真实描述和质量-数量权衡，但局限于COCO类别。MME（能力评估，2.4K问题）包含14个子任务，全面但仅yes/no格式。

我们采用三维评估策略（POPE + CHAIR + MME），分别测量判别准确率与偏差检测、生成质量与详细度权衡、14个任务的通用能力保持。

DPO-only模型的悖论（第4.3节）展示了多维评估的必要性：POPE F1=0.900（优秀，暗示成功），CHAIR_i=31.83%（差，揭示失败），MME=1964.5（中等，显示知识损失）。单一基准评估不足以验证模型性能，需要多维度联合验证。

## 2.4 参数高效微调：LoRA

### 2.4.1 低秩适应理论

全量微调大模型（7B+参数）成本高昂：需要存储完整梯度和优化器状态，每个checkpoint保存整个模型，更新所有参数。Hu等人（2021）的LoRA（Low-Rank Adaptation）通过低秩分解简化权重更新。

核心思想是神经网络权重在微调过程中的更新具有低秩特性：W_adapted = W_pretrained + ΔW，其中ΔW ≈ BA（低秩分解）。B ∈ R^(d×r)为下投影矩阵，A ∈ R^(r×k)为上投影矩阵，秩r << min(d, k)（通常4-32）。

前向传播变为h = Wx + BAx = Wx + ΔW·x。可训练参数从全量微调的d × k降至LoRA的(d + k) × r，约为原始参数的0.1-1%。以Qwen3-VL-8B配置LoRA r=8为例：总参数7.62B，可训练参数22M，占比0.29%。

### 2.4.2 LoRA超参数

秩r决定了容量。小秩（4-8）训练快、显存低但可能欠拟合，大秩（64-128）接近全量微调、容量高。经验法则是r=8-16足以应对大多数任务。

Alpha（α）作为缩放因子：ΔW_scaled = (α/r) · BA，通常设置α = 2r（如r=8时α=16），实际上控制了有效学习率。

目标模块可以选择性地仅包含注意力Q、V（LLaMA原始方法），或全面覆盖所有线性层（我们的方法）。这是覆盖范围与参数数量的权衡。

### 2.4.3 LoRA变体

LoRA+（Hayou et al., 2024）为A和B矩阵设置不同学习率，lr_B = lr_A × λ（通常λ=16），原因是B初始化为零、A随机，导致重要性不对称。

DoRA（Weight-Decomposed LoRA，Liu et al., 2024）将权重分解为幅度和方向：W = m · (W_0 + BA) / ||W_0 + BA||，性能更好但显存需求是LoRA的2倍。

QLoRA（Dettmers et al., 2023）结合LoRA与4-bit量化，在单个48GB GPU上实现65B模型训练，代价是量化带来的轻微精度损失。

我们使用标准LoRA的原因是：8B模型可适配40GB A100，DoRA/LoRA+在此规模下收益微小（文献报告<1%提升），标准LoRA得到LLaMA-Factory框架广泛支持，我们专注于幻觉缓解而非参数效率优化。

### 2.4.4 文献中的LoRA秩消融

不同研究的最优秩不同。Hu等人（2021）在GPT-3语言任务上发现r=4-8最优，Ding等人（2023）在LLaMA-7B指令任务上发现r=16最优，Zhang等人（2023）在BERT自然语言理解上发现r=8最优。

我们的发现（第5.1节）是LoRA秩对幻觉缓解影响极小：r=4达到POPE F1=0.882和CHAIR_i=16.59%，r=8为F1=0.855和16.64%，r=16为F1=0.876和17.07%，r=32为F1=0.873和16.10%。POPE/CHAIR方差小于2%。因此我们使用r=8作为默认配置（容量充足，22M可训练参数，广泛采用的基线）。

## 2.5 现有文献的空白

尽管取得了显著进展，但几个问题仍未得到充分探索。

### 2.5.1 SFT数据规模效应

文献通常假设"数据越多越好"（LLaVA使用665K，LLaVA-RLHF使用400K SFT数据）。但未探索的问题包括：SFT数据规模是否呈现单调改善？幻觉缓解与通用能力的最优平衡点是什么？更大的SFT数据集是否会放大分布偏差（yes-bias）？

我们通过系统消融{5K, 10K, 25K, 50K}揭示了"少即是多"现象（第5.2节）：5K达到POPE F1=0.922（最优），50K下降至F1=0.855（-6.7pp）。机制分析显示更大数据集放大了正例样本比例，从而加剧yes-bias。

### 2.5.2 DPO Beta敏感性

多数论文使用单一beta值（HA-DPO: 0.5，RLHF-V: 0.5-1.1）。未探索的问题包括：beta的完整谱（特别是β > 0.5）、崩溃阈值（最小可行β）、POPE与CHAIR在beta范围内的权衡曲线。

我们的6点消融{0.01, 0.05, 0.1, 0.2, 0.5, 1.0}（第5.3节）发现：β < 0.1导致模型崩溃（yes-ratio → 0），β = 1.0达到最优平衡（F1=0.846，CHAIR_i=22.04%）。

### 2.5.3 VLM后训练中的知识灾难性遗忘

文献通常整体测量通用能力保持（单一MME分数）。未探索的问题包括：哪些能力最脆弱（感知 vs 认知）、细粒度任务级分析（14个MME子任务）、DPO能否恢复SFT导致的知识损失。

我们首次系统量化了知识灾难性遗忘（第6章）：SFT损害名人识别（-7.35pp）、艺术品（-7.00pp）、地标（-6.75pp），DPO恢复名人识别（+2.65pp超越基线）。我们提出了6维幻觉框架（存在性、属性、计数、知识、空间、OCR）。

### 2.5.4 DPO-only与顺序流程

所有先前工作使用SFT → DPO流程。未探索的问题包括：SFT是否必要，或DPO能否单独工作？每个阶段在幻觉缓解中的作用是什么？

我们的DPO-only消融揭示了判别-生成分离（第4.3节）：DPO-only达到POPE F1=0.900（最优）但CHAIR_i=31.83%（最差）。结论是SFT对生成质量至关重要，DPO用于判别准确率。

## 2.6 本章小结

本章回顾了VLM幻觉缓解研究的四个支柱：缓解方法（训练式方法如RLHF-V、HA-DPO、LRV在部署效率上优于后处理方法如Woodpecker、VCD）、偏好学习（DPO因稳定性优于RLHF，beta和训练轮数是关键超参数）、评估方法（三维评估POPE + CHAIR + MME能够捕获判别、生成和能力维度）、参数效率（LoRA r=4-8足以满足VLM微调，更高秩收益微小）。

我们的工作通过全面消融（20个模型配置）解决了四个文献空白，发现了"少即是多"的SFT数据规模规律、最优DPO β=1.0（比先前0.1-0.5范围更广）、知识灾难性遗忘的量化、DPO-only悖论揭示流程必要性。这些发现深化了对SFT+DPO动态机制的理解，为实践者提供了可操作的指南。

表2.1总结了我们与最新技术的对比。我们的True Optimal模型（5K SFT + 5.7K DPO，β=1.0单轮）达到POPE F1=0.889、CHAIR_i=20.12%（领域最优）和MME CPR=99.1%（能力保持率），核心创新是"少即是多"的SFT规模和β=1.0的DPO配置。SFT 5K单独模型达到POPE F1=0.922（最高判别准确率），验证了数据规模发现。DPO-only模型展示了判别-生成悖论（POPE F1=0.900但CHAIR_i=31.83%），证明了两阶段流程的必要性。
