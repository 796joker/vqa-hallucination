## 图7.1：案例研究1 - 存在性幻觉修正

**使用章节**：第7.2节，案例研究1

**提示词**：
```
创建一个学术论文图表，展示视觉问答案例研究：

【布局】1400×900像素，横向分割
- 左侧40%：客厅场景照片
  · 包含：棕色沙发、木质茶几、台灯、书架
  · 不包含：电视机（重点！）
  · 风格：温馨现代客厅，自然光

- 右侧60%：纵向分为4个面板

【顶部面板】高度150px，浅蓝色背景(#e3f2fd)
📋 问题："这张图片中有电视机吗？"
✅ 正确答案：否（POPE对抗性分割测试）

【面板1】高度200px，浅红色背景(#ffebee)
🤖 **Base模型** (Qwen3-VL-8B)
💬 回答："是的，客厅里有一台电视机。"
❌ 判定：存在性幻觉（电视实际不存在）
🔍 原因：过度依赖客厅-电视共现先验

【面板2】高度200px，浅橙色背景(#fff3e0)
🤖 **SFT 50K模型**
💬 回答："是的，墙上似乎有一台电视。"
❌ 判定：仍然幻觉（SFT放大yes-bias）
📊 Yes-ratio: 0.521 (+9pp偏移)

【面板3】高度200px，浅绿色背景(#e8f5e9)
🤖 **True Optimal模型**
💬 回答："否，图像中没有看到电视机。"
✅ 判定：正确（DPO修正判别偏差）
📊 Yes-ratio: 0.413 (接近理想0.50)

【样式要求】
- 字体：思源黑体或微软雅黑，12pt正文
- 图标：彩色emoji增强可读性
- 分隔线：各面板间2px白色边距
- 圆角：所有面板8px圆角

【顶部标题】
"案例研究1：存在性幻觉修正 - POPE对抗性分割"
字体：16pt黑体加粗，居中
```

## 使用说明

### 推荐AI工具
1. **Gemini 2.0**（推荐）：理解复杂布局，生成学术风格图表
2. **DALL-E 3**：高质量渲染，适合照片场景
3. **Midjourney**：艺术风格优秀，适合案例研究1和3

### 生成步骤
1. 复制上述提示词到AI工具
2. 调整参数：
   - 分辨率：1400×900或1920×1080
   - 风格：academic diagram, technical illustration
   - 格式：PNG with transparency
3. 检查生成结果：
   - 文字是否清晰可读（中文支持）
   - 颜色编码是否正确（红色=错误，绿色=正确）
   - 布局是否符合40-60分割
4. 后处理（可选）：
   - 使用Photoshop/GIMP调整文字大小
   - 添加标注箭头或高亮框

### 保存规范
- 文件名：`案例研究{编号}_{主题}.png`
  - 案例研究1_存在性幻觉.png
  - 案例研究2_Yes-Bias.png
  - 案例研究3_知识遗忘.png
  - 案例研究4_计数准确性.png
  - 案例研究5_保守性.png
  - 案例研究6_DPO-only悖论.png
- 保存位置：`报告/图表/`

### 插入到报告
生成后在第7章对应位置插入：
```markdown
![图7.1：案例研究1 - 存在性幻觉修正](../图表/案例研究1_存在性幻觉.png)

**图7.1**：Base模型和SFT 50K都错误判断"有电视"，True Optimal正确判断"无电视"，证明DPO修正了共现偏差。
```
